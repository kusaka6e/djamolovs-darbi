API

Saīsinājums API apzīmē "Application Programming Interface" (lietojumprogrammu saskarne).
Lielākā daļa lielo uzņēmumu kādā posmā izstrādā API klientiem vai iekšējai lietošanai.
Lai saprastu, kā un kā API tiek izmantota attīstībā un uzņēmējdarbībā, vispirms ir jāsaprot, kā darbojas globālais tīmeklis.
